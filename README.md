# Exames na Mão

Aplicativo para agendamento de exames ocupacionais e consultas médicas em Ipatinga-MG.

## Sobre o Projeto

O **Exames na Mão** conecta usuários a clínicas da região, permitindo:

- Buscar exames ocupacionais (admissional, demissional, toxicológico, etc.)
- Agendar consultas médicas
- Comparar preços entre clínicas
- Agendar diretamente via WhatsApp

## Tecnologias

Este projeto foi construído com:

- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn/UI
- Framer Motion

## Como Executar

```bash
# Instalar dependências
npm install

# Executar em modo de desenvolvimento
npm run dev

# Build para produção
npm run build
```
